O Portfólio é o projeto final do curso de Front-End, desenvolvido com o objetivo de reunir e apresentar meus projetos, habilidades e experiências de forma clara e visualmente atraente. Este projeto demonstra conhecimentos em HTML, CSS, JavaScript e boas práticas de desenvolvimento web.

Funcionalidades Apresentação pessoal com informações sobre mim. Exposição de projetos desenvolvidos durante o curso. Contato através de formulário funcional. Layout responsivo, adaptável para dispositivos móveis e desktops. Navegação intuitiva e interativa.

Tecnologias Utilizadas HTML5 – Estrutura semântica das páginas. CSS3 – Estilização e design responsivo. JavaScript – Interatividade e manipulação do DOM.
